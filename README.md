
Projeto desenvolvido para a disciplina Dinamismo na Web

Objetivo da Atividade >

Desenvolver uma aplicação web simples utilizando HTML, CSS e JavaScript, realizando consumo de API com fetch e manipulação dinâmica da interface.

O que foi feito >

Um mural de notícias que carrega usuários e posts da API pública JSONPlaceholder, com interface para adicionar, editar e remover posts localmente.
